Data Analysis


Data is a valuable asset which should be harvested to its full potential.

Why Analyse data?

Data analysis helps us with
  • Unearthing useful information
  • Answering questions

Data Analysis Steps

  1. Organize data : context
  2. Format and adjust data : filtering, sorting
  3. Transform : change data type, merge, aggregate
  4. Validate : checking and rechecking the quality that it is complete, accurate, secure and consistent

Data Context: a set of conditions (an environment) related to the data.


Type of Data

Structured Data

Structured Data has a well-defined structure or adheres to a specified data model, can be stored in well-defined schemas such as databases, and in many cases can be represented in a tabular manner with rows and columns.

Semi-Structured Data

has some organizational properties but lacks a fixed or rigid schema.

Unstructured Data

does not have an easily identifiable structure, cannot be organized in a mainstream relational database in the form of rows and columns.


Data Formats

Continuous

Data that is measured and can have almost any numeric value

Discrete

Data that is counted and has a limited number of values

Qualitative

Subjective and explanatory measures of qualities and characteristics

Quantitative

Specific and objective measures of numerical facts

Nominal

A type of qualitative data that isn’t categorized with a set order

Ordinal

A type of qualitative data with a set order or scale

Structured

Data organized in a certain format, like rows and columns

Unstructured

Data that isn’t organized in any easily identifiable manner


General issues with data

  • Incomplete and/or
  • Incorrect and/or
  • Irrelevant

We refer to data with the above issues as "Dirty Data"
Common issues of Dirty Data are
  • Spelling errors
  • Inconsitent labels
  • Formats
  • Lengths
  • Missing
  • Duplicate
  • Outdated
  • Incorrect/Inaccurate
